About This Page

This is a demonstration of a sidebar that switches position based on the selected language. Click the language toggle button in the top right corner to switch between English and Arabic layouts.

Features

The sidebar automatically moves from left to right when switching to Arabic (RTL layout), and the entire page direction adjusts accordingly. This provides a better user experience for Arabic-speaking users.

Responsive Design

The layout uses smooth transitions to create a seamless experience when switching between languages. All elements are properly aligned and maintain their functionality in both language modes.
